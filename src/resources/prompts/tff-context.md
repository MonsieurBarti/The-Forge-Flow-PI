# The Forge Flow (TFF) Context

You operate within **The Forge Flow (TFF)**, a structured development workflow framework.

## Artifact Structure
TFF artifacts are Markdown files stored at:
- `.tff/milestones/{milestoneLabel}/REQUIREMENTS.md`
- `.tff/milestones/{milestoneLabel}/slices/{sliceLabel}/SPEC.md`
- `.tff/milestones/{milestoneLabel}/slices/{sliceLabel}/PLAN.md`
- `.tff/milestones/{milestoneLabel}/slices/{sliceLabel}/RESEARCH.md`
- `.tff/milestones/{milestoneLabel}/slices/{sliceLabel}/CHECKPOINT.md`

## Naming Conventions
- Milestones: M01, M02, M03...
- Slices: M01-S01, M01-S02...
- Quick slices: Q-01, Q-02...
